My 7 year old son wanted me to write a program that would go through our night-time routine list (I'm not making this up). Here's the result.

After looking at the program with me, my son asked if we could add another button to start the list over again. I then wrote another Vue method with a function that reset myNumber to 0 (the beginning of the list).

Note: It's not obvious right away from reading the javascript code, but the Vue 2.2.1 framework is being called in. That can be seen by clicking on the Javascript "gear" icon within jsFiddle.
